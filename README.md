# Olympiad Talent Identification Test

This repository contains a collection of olympiad-style problems designed for a talent identification test for middle school students (grades 7–9).

The goal of this project was to introduce students to olympiad-style problem solving and evaluate creative reasoning abilities rather than routine calculations.

## Topics Covered

The problems explore concepts commonly appearing in computer science and mathematics olympiad preparation, including:

- Combinatorics
- Number Theory
- Logical Reasoning
- Algorithmic Thinking
- Graph-based reasoning

Several problems were inspired by well-known olympiad-style puzzles and were simplified to make them accessible for younger students.

## Original Contributions

Problems **6** and **7** in the test were originally designed by me as part of this project.

## Project Context

This test was created as part of an initiative aimed at introducing younger students to olympiad-style thinking and identifying students with strong analytical and problem-solving potential.

Although the broader educational program did not continue, the project provided valuable experience in problem design and mathematical assessment.
